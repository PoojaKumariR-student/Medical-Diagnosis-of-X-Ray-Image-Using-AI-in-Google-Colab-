# Medical-Diagnosis-of-X-Ray-Image-Using-AI-in-Google-Colab-
📝 DESCRIPTION :

The project focuses on developing an Artificial Intelligence-based medical diagnosis system for analyzing Chest X-Ray images using Deep Learning techniques. The system utilizes Convolutional Neural Network (CNN) architectures such as ResNet and VGG to automatically classify chest X-Ray images and detect lung-related diseases with high accuracy. Chest X-Ray analysis is widely used in healthcare for diagnosing diseases such as pneumonia, lung infections, COVID-19, and other respiratory abnormalities. Manual interpretation by radiologists can be time-consuming and may vary depending on expertise. The proposed AI system automates the diagnosis process, improves prediction accuracy, and reduces human effort. The project also integrates Grad-CAM (Gradient-weighted Class Activation Mapping), an Explainable AI technique that visually highlights the important regions in the X-Ray image influencing the model’s prediction. This improves model transparency and helps understand how the CNN model makes diagnostic decisions.The entire system is implemented using Python in Google Colab and demonstrates the effectiveness of CNN-based deep learning models for intelligent chest disease detection.
